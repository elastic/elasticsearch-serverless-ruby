This is the official Ruby client for the **Elasticsearch Serverless** service. 

If you want to develop your Ruby application with the Elasticsearch Stack, refer 
to the [Elasticsearch Client](https://github.com/elastic/elasticsearch-ruby) 
instead. 

If you want to develop your Ruby application with Elastic Enterprise Search, 
refer to the 
[Enterprise Search Client](https://github.com/elastic/enterprise-search-ruby/).
